Standardised buttons. Can be applied to anchors, inputs and buttons.

c-notification

A notification/alert component. Currently has "error" and "success" modifiers.

You are using an outdated browser. Please upgrade your browser to improve your experience. 
<div class="c-notification c-notification--error">
  You are using an <strong>outdated</strong> browser. Please <a href="http://browsehappy.com/" class="c-notification__link">upgrade your browser</a> to improve your experience.
</div>

c-progress

A scalable progress indicator component. If N is the number of "c-progressitem" elements, then the number of "c-progresspath__bar" elements is N - 1.

1
First step
2
Second step
3
Third step
4
Fourth step
